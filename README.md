# CodeAlpha_LanguageLearningApp
A multi-lingual language learning dashboard tracking vocabulary data for Tamil, Japanese, and Spanish. Features dynamic 3D card-flipping mechanics, deck navigation matrices, and an integrated real-time recall testing engine. Built with HTML5, CSS3, and Vanilla JavaScript.
# 🌐 LingoVerse - Multilingual Language Hub

An advanced, fully-responsive Language Learning application built for the **CodeAlpha App Development Internship** (Task 4). 

This app serves as an interactive multi-lingual vocabulary hub allowing users to switch dynamically between three distinctly structured languages: **Tamil (தமிழ்)**, **Japanese (日本語)**, and **Español (Spanish)**.

## 🚀 Interactive Features
* **Dynamic Dataset Swapping:** Seamlessly switch between regional dialects with instantaneous tab navigation tracking.
* **Complex 3D Animation Engine:** Hardware-accelerated CSS properties enable natural perspective card flipping to view term definitions.
* **Synchronized Application State:** Card navigation controls keep the active vocabulary selection perfectly mapped to both the study flashcard and the interactive quiz section.
* **Instant Evaluation Feedback:** Automated choice auditing highlights user selections with contextual green/red visual indicators based on correctness.

## 🛠️ Tech Stack
* **HTML5:** Modular semantic structuring for layout containers, selector headers, and testing grids.
* **CSS3:** Custom linear background blending, CSS custom variables (`:root`), absolute container positioning, and complex `preserve-3d` animations.
* **JavaScript (ES6):** State database dictionaries, runtime click listener event binding, string mappings, and modular dynamic element injection.
